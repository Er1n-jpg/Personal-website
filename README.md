# Personal-website
hey guys! For the athena award I made a personal website, it looks really crusty dusty and musty but it was pretty much my very first experience within web development aside from a scrapyard chrome extension project. 

The very first 8 hours of the project was made for the athena award back in november and i'm shipping the rest of the hours for campfire flagship!

This website includes a main page, a page about my hobbies and interests and a blog page, its main purpose is to mostly just showcase myself and my projects to other developers. It doesn't look like very much but
it's my very first website project!
